Suggested Reading
-----------------
* FASTA
  - [FASTA paper](http://www.ncbi.nlm.nih.gov/pubmed/2983426)
  - [FASTA web search at EBI](http://www.ebi.ac.uk/Tools/sss/fasta/)

* BLAST
  - [Original BLAST paper](http://www.ncbi.nlm.nih.gov/pubmed/2231712)
  - [BLAST Results Explained](ftp://ftp.ncbi.nlm.nih.gov/pub/factsheets/HowTo_NewBLAST.pdf)
  - [Karlin-Altschul Statistics](http://oreilly.com/catalog/blast/chapter/ch04.pdf)
  - [The Statistics of Sequence Similarity Scores](http://www.ncbi.nlm.nih.gov/blast/tutorial/Altschul-1.html) from NCBI
  - [Running BLAST from the command line](http://www.ncbi.nlm.nih.gov/books/NBK1763/)
  - [BLAST in the CLOUD](http://www.youtube.com/watch?v=wLm-RfdcvnU) NCBI tutorial on Amazon Web Services

* BLASTPLUS
  - [BlastPlus at NCBI](https://blast.ncbi.nlm.nih.gov/Blast.cgi?PAGE_TYPE=BlastDocs&DOC_TYPE=Download)

* Receiver Operator Characteristic (ROC)
  - [SciPy ROC example](http://scikit-learn.org/0.11/auto_examples/plot_roc.html)
  - [Drawing ROC Curve](http://docs.eyesopen.com/toolkits/cookbook/python/plotting/roc.html)

* [Research Computing Center](https://rcc.uchicago.edu/)
  - [User Guide](https://rcc.uchicago.edu/docs/)
  - [Tutorials](https://rcc.uchicago.edu/docs/tutorials/index.html)

* [RCC Utilities Repository with Demos](https://github.com/uchicago-bio/RCC-Utilities)
  - Run a blastp search on a single node [single-node-blast.sh](https://raw.githubusercontent.com/uchicago-bio/RCC-Utilities/master/single-node-blast.sbatch)
